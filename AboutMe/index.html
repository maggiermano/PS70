About Me

Why am I taking PS70?

The short answer is that I love problem solving. Developing an idea into a project that solves a problem whether big or small is probably one of my favorite things to do. As a CS major I have put this into practice already by completing websites or applications that have a distinct purpose. This class allows me to learn cool skills which I can eventually use to make a tangible project that solves some sort of problem which is why I was very excited to enroll in it. The smaller weekly projects seem fun too.
